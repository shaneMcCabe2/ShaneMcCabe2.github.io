## Featured Projects

### SQL Analysis & Practice Tool

**Skills:** Python, BigQuery, dbt, Streamlit, Claude API, Data Engineering, NLP, Statistical Analysis

Analyzed 300k+ Stack Overflow posts (2020-2022) to surface real-world SQL patterns across dialects, error types, and query complexity. Built an end-to-end pipeline from raw public dataset ingestion through dbt modeling to a deployed Streamlit dashboard with an AI-powered practice question tool.

**Key Findings:**
- Dialect trends: Identified shift in community focus toward BigQuery and Snowflake syntax post-2021
- Error patterns: Clustered most common SQL mistakes by type, frequency, and vote signal
- Complexity scoring: Built a query scoring model based on JOIN depth, subquery nesting, and window function usage

**What's built:**
- dbt models across staging and marts layers on BigQuery
- Analysis layer covering dialect frequency, error patterns, keyword trends, and complexity scoring
- AI practice question generator using Claude API, sourcing questions from real SO content weighted by community votes and complexity tier

### Data Analyst Job Market Analysis

**Skills:** Python, Plotly, Pandas, Git, LLM, Statistical Analysis, Data Visualization, Data Engineering

Analyzed 45,000+ job postings to identify skill demand, salary drivers, and career progression patterns across data analyst roles.

**Key Findings:**
- Experience is the strongest salary predictor (r=0.332)
- Top salary-correlated skills: Python (+$9,608), Snowflake (+$13,252), AWS (+$16,952)
- Python maintains salary correlation even after controlling for experience level
- Entry to Senior progression yields ~46% salary increase ($83K to $121K)
